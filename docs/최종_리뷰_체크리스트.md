# 성경 1일1독 앱 최종 리뷰 체크리스트

## 1. Supabase 연동

| 항목 | 상태 | 비고 |
|------|------|------|
| bible_completed_days 테이블 | ✅ | docs/SUPABASE_COMPLETED_DAYS.md 참고 |
| completedDays 로드 (로그인 시) | ✅ | loadFromSupabase |
| completedDays 동기화 (저장 시) | ✅ | syncAllToSupabase, 2.5초 디바운스 |
| 테이블 미존재 시 | ✅ | 앱 정상 동작, completed_days만 스킵 |

## 2. 1일1독 완료 & 진도율 기능

| 항목 | 상태 | 비고 |
|------|------|------|
| 제 N일 오른쪽 완료 체크 | ✅ | BibleDaily.tsx |
| 홈 1일1독 진도율 버튼 | ✅ | 완료 1건 이상 시 표시 |
| 진도율 페이지 (게이지) | ✅ | 전체 % + N일 완료 |
| 진도율 페이지 (달력) | ✅ | 월별 완료 날짜 표시 |
| 달력 월 이동 | ✅ | ‹ › 버튼 |

## 4. 성능 개선

| 항목 | 적용 |
|------|------|
| 동기화 디바운스 | 2.5초 (과도한 API 호출 방지) |
| 달력 셀 key | `${date}-${idx}` (중복 방지) |
| Lazy loading | 모든 페이지 lazy import |
| Virtual list | BibleBookViewer (TanStack Virtual) |

## 5. 테스트 시나리오

1. **비로그인**
   - 홈 → 시작하기 → 책 선택 → 1일1독 읽기
   - 제 1일 오른쪽 완료 체크
   - 홈에 "1일1독 진도율" 버튼 표시 확인
   - 진도율 페이지: 게이지, 달력 확인

2. **로그인**
   - 로그인 후 완료 체크
   - 2.5초 후 Supabase 동기화 (Network 탭 확인)
   - 다른 기기/탭에서 로그인 → 완료 데이터 로드 확인

3. **성경책 보기**
   - 성경책 보기 → 한국어 기본 확인
   - 검색, 스크롤 정상 동작

4. **일기장**
   - 메모, 찜, 받은 말씀 탭 전환
   - 날짜 필터, 검색

## 6. 알려진 제한사항

- bibleText 청크 ~2.3MB: 성경책/말씀 읽기 첫 진입 시 로딩
- 동기화: 전체 replace 방식 (데이터 많으면 수 초 소요 가능)
